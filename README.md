# Gaussian Elimination

## AIM:
To write a program to find the solution of a matrix using Gaussian Elimination.

## Equipments Required:
1. Hardware – PCs
2. Anaconda – Python 3.7 Installation / Moodle-Code Runner

## Algorithm
1. Read the augmented matrix of the given system of linear equations.
2. Convert the matrix into upper triangular form using row operations
3. Apply back substitution to find the values of unknown variables.
4. Display the solution and stop the program.

## Program:
```
'''Program to solve a matrix using Gaussian elimination without partial pivoting.
Developed by:MANUSIYA R 
RegisterNumber: 25016729
'''
import numpy as np
import sys 
n=int(input())
a=np.zeros((n,n+1))
x=np.zeros(n)
for i in range (n):
      for j in range(n+1):
           a[i][j]=float(input())
for i in range(n):
      if a[i][j] == 0:
         sys.exit('Divide by Zero detected!')
      for j in range(i+1,n):
         ratio=a[j][i]/a[i][i]
         for k in range(n+1):
            a[j][k]=a[j][k]-ratio*a[i][k]
x[n-1]=a[n-1][n]/a[n-1][n-1]
for i in range(n-2,-1,-1):
       x[i]=a[i][n]
       for j in range(i+1,n):
            x[i]=x[i]-a[i][j]*x[j]
       x[i]=x[i]/a[i][i]
for i in range(n):
      print('X%d = %0.2f'%(i,x[i]),end = ' ')
```
